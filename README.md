# Handle Stalker

## Testing

###### Chrome

1. Clone or [download](https://github.com/samanthakem/HandleStalker/archive/master.zip) project;

2. Navigate to `chrome://extensions`

3. Click on `Load unpacked extension...`

4. Select the `extension` folder
